## 一：项目介绍
1，本项目用到的技术是Vue3，TS，Vite，Element-Plus。参照的是vue3-element-admin企业级后台管理系统。主要用于个人学习和练手。

## 二：项目启动

```bash
# 克隆代码
git clone https://github.com/yuanjuan/my-vue3-element-admin.git

# 切换目录
cd my-vue3-element-admin

# 安装依赖
npm install

# 启动运行
npm run dev
```
